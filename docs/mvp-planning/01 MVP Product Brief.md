# 01 MVP Product Brief

## Product name

Working name:

> Construction WhatsApp Reporting SaaS

The name can change later.

## Product vision

Build a SaaS platform for small construction companies to capture daily site data through WhatsApp and review/export that data through a dashboard.

## Target customer

Small contractors and builder-developers managing multiple projects, teams, and site updates.

## Primary users

- Company owner
- Project manager
- Site engineer
- Site supervisor
- Store/material incharge
- Admin/back-office user

## Core problem

Construction site information is usually scattered across WhatsApp chats, calls, notebooks, and Excel sheets.

This creates problems:

- difficult to know what happened on a given day
- hard to verify work with images
- material received/issued is poorly tracked
- manpower data is inconsistent
- reports take manual effort
- owners lack date-wise visibility

## MVP goal

Create a reliable first version that allows companies to:

- record progress through WhatsApp
- record manpower through WhatsApp
- record material received and issued through WhatsApp
- attach images as proof
- view all data in a dashboard
- filter by project and date range
- export to Excel
- receive automatic daily WhatsApp summaries
- review project analytics such as plan vs actual, manpower distribution, area-wise progress, and material stock

## MVP positioning

This is not a full ERP.

This is not a full project management tool.

This is a daily site reporting and visibility platform.

## Key product principle

WhatsApp is for data capture.

Dashboard is for review, management, filtering, analytics, and export.
