# 10 Pilot Checklist

## Product readiness

- [ ] Company can be created.
- [ ] Platform admin can create company and initial users.
- [ ] Company owner/admin can add more users.
- [ ] Admin can be created.
- [ ] Users can be added.
- [ ] Projects can be created.
- [ ] Users can be assigned to projects.
- [ ] Project locations can be defined.
- [ ] Project knowledgebase templates are available.
- [ ] Schedule can be uploaded.
- [ ] Scope of work can be uploaded.
- [ ] BOQ can be uploaded.
- [ ] Resource plan can be uploaded.
- [ ] User-role list can be uploaded.
- [ ] Areas and sub-areas can be uploaded.
- [ ] Units can be uploaded.
- [ ] Activities or schedule activities can be uploaded.
- [ ] Uploaded project data is used for validation.

## WhatsApp readiness

- [ ] WhatsApp provider decision is documented for pilot.
- [ ] WhatsApp provider webhook is configured.
- [ ] Architecture does not permanently depend on Twilio.
- [ ] Meta WhatsApp Cloud API webhook can be incorporated later without redesign.
- [ ] User can start naturally without special commands.
- [ ] User is recognized by phone number.
- [ ] User can select project.
- [ ] User can record progress.
- [ ] User can record manpower.
- [ ] User can record material received.
- [ ] User can record material issued.
- [ ] User can upload proof image.
- [ ] User can send voice note.
- [ ] Voice note is transcribed.
- [ ] Assistant asks only for missing information.
- [ ] Assistant summarizes and confirms before saving.
- [ ] Hindi/Hinglish examples work.
- [ ] Assistant handles offensive or irrelevant messages professionally.
- [ ] Daily WhatsApp summary is sent automatically.
- [ ] Daily WhatsApp summary defaults to 7:00 PM local project time.
- [ ] Daily WhatsApp summary time is configurable.

## Dashboard readiness

- [ ] Admin can log in.
- [ ] Admin can view project data.
- [ ] Admin can filter by date.
- [ ] Admin can filter by date range.
- [ ] Admin can select one project or multiple projects.
- [ ] Admin can view progress entries.
- [ ] Admin can view manpower entries.
- [ ] Admin can view material transactions.
- [ ] Admin can view material stock.
- [ ] Admin can view plan vs actual.
- [ ] Admin can view total progress till date by area.
- [ ] Admin can view manpower distribution chart for last week/month.
- [ ] Admin can view tomorrow's activities.
- [ ] AI subscribers can view AI insights.
- [ ] Admin can view linked images.
- [ ] Admin can export Excel.
- [ ] Admin can upload project knowledgebase.
- [ ] Admin can see template validation errors.
- [ ] Admin can view WhatsApp provider settings.
- [ ] Admin can view or configure AI key mode.

## Data readiness

- [ ] PostgreSQL is set up.
- [ ] Backups are configured.
- [ ] Object storage is configured.
- [ ] Audit logs are captured.
- [ ] Sensitive data is not logged unnecessarily.

## Security readiness

- [ ] No hardcoded credentials.
- [ ] WhatsApp webhook validation enabled where supported by provider.
- [ ] AI keys are stored securely.
- [ ] AI keys are not printed in logs.
- [ ] Strong admin passwords.
- [ ] Role permissions enforced.
- [ ] Company data isolation tested.

## Pilot operations

- [ ] Pilot company selected.
- [ ] Pilot projects configured.
- [ ] Pilot users onboarded.
- [ ] WhatsApp instructions shared.
- [ ] Support contact defined.
- [ ] Daily review process agreed.
- [ ] Feedback tracker created.
